website
=======

to run:
 - install roots: `sudo npm install roots -g`
 - clone this repo `git clone https://github.com/core2062/website.git`
 - cd into the directory of the repo `cd website`
 - run `npm install`
 - run `sudo npm link roots`
 - run `roots watch`
